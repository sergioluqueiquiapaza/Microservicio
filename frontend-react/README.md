# React + Vite

## Instalación

### 1. Clonar el repositorio
```bash
git clone [URL_DEL_REPOSITORIO]
cd [NOMBRE_DEL_PROYECTO]
```

### 2. Instalar dependencias del frontend
```bash
npm install
```

### 3. Instalar dependencias adicionales necesarias
```bash
# Bootstrap y sus dependencias
npm install bootstrap@5.3.2 react-bootstrap

# Enrutamiento (para navegar entre páginas)
npm install react-router-dom

# Para hacer peticiones HTTP al backend
npm install axios

# Para manejo de tokens JWT
npm install jwt-decode

# Iconos
npm install react-icons

# Para formularios y validaciones
npm install formik yup
```

### 4. Ejecutar el proyecto
```bash
npm run dev
```

## PREPARACIÓN DEL ENTORNO Y DISEÑO

### Dependencias Instaladas:
- **Bootstrap 5.3.2 y React-Bootstrap**: Para estilos y componentes UI
- **React Router DOM**: Sistema de enrutamiento
- **Axios**: Cliente HTTP para comunicación con el backend
- **JWT Decode**: Decodificación de tokens de autenticación
- **React Icons**: Librería de iconos
- **Formik y Yup**: Manejo y validación de formularios
